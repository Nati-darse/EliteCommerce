# Auth Service API

## Purpose

Authentication API built with Express and Supabase.

## Features

* User Signup
* User Login
* Input Validation using Zod
* Rate Limiting
* Error Handling Middleware

## Tech Stack

* Node.js
* Express
* Supabase
* PostgreSQL
* Zod

## Installation

npm install

## Environment Variables

SUPABASE_URL=
SUPABASE_ANON_KEY=
PORT=5000

## Run

npm run dev

## API Endpoints

POST /signup

POST /login

GET /health
